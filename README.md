# Year progress bar

This project generates a bar of elapsed time (in minutes) in the year

## Run Locally

Clone the project

```bash
  git clone https://github.com/MrPotatoXx/progress_bar_year
```

Go to the project directory

```bash
  cd progress_bar_year
```

Install tqdm

```bash
  pip install tqdm
```

Start the proyect

```bash
  python progressbar.py
```
